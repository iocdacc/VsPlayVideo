简介

1.暂时只支持网易云音乐之后添加本地音频

2.暂时不支持手机端（未测试）

3.版本初期可能会有BUG

4.W3C关于自动播放的政策，Chrome v66版本开始限制了单位时间内打开网页时自动播放的次数。超过次数时自动播放会报错，手动开始播放正常。

使用方法

                <head>

<script src="./dist/vsPlayAudio.min.js"></script>

</head>
<body>

...

<div id="vsPlayAudio"></div>

...

new vsPlayAudio({
    element: "vsPlayAudio",
    autoPlay: false,//自动播放
    music: {
        //必需项,音乐配置
        type: 'cloud',
        //必需项,网易云方式指定填'cloud'
        source: 2778782119
        //必需项,网易云音乐歌单id
        //登录网易云网页版,在网页地址中拿到
        // ... playlist?id=2778782119
    },
    random: false//随机播放
});
</body>
